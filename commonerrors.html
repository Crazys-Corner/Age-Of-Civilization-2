Common Errors and their Fixes




Troubleshooting Server to Client connection issues


If you are having an issue connecting to the server, attempt to restart your pc, router, and modem. If that does not work, open a ticket in the discord. You can open tickets from the #help channel.

Missing Mods or Mod Conflicts


If you are getting errors printed in red text when joining the server talking about missing or conflicting mods, your modpack is not up to date. To update your modpack, see the following tutorial:
Tutorial (External Website)

Eating Food is Slow


This is a feature of the “Spice of Life” mod. See its mod page for more details:
Modpage (External Website)

Game disconnected or crashed whilst flying

This is an unfortunate issue regarding the max packet size servers can send to players. You were rendering so many chunks that the server could not send you the data fast enough. If this happens open a ticket in the discord #help channel immediately and please do not join until given the go ahead by a staff member.

Server Closed whilst flying

This is an unfortunate issue caused by loading lots of chunks with vehicles from the Immersive Vehicles mod. We are still looking into the exact cause of the issue, but for now please see the Game Crashed/Disconnected When Flying article for help.

JEI not showing Recipies

This is not a glitch or bug, you have yet to research the item in the tech tree. See more info about the tech tree here (External Website.)
Important Note: Do not join an FTB quests team under any circumstance, you must create your own.

Cannot see any ores

This is an intentional feature. If you do not have the Miner role you will not be able to see ores. If you are a miner and you can't see ores, please make a ticket.
Please see Skill System for more details.
